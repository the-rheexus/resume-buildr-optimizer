# Naming Convention

## Slugging

- `{company_slug}` — full company name, lowercased, spaces → underscores
  (`Boston Consulting Group` → `boston_consulting_group`).
- `{role_slug}` — short, distinguishing role, lowercased, spaces → underscores.
  Not the full posting title (`Consultant, Experienced Hire, United States` →
  `consultant`). If Josh applies to multiple roles at the same company, add a
  distinguishing word (`consultant_experienced` vs. `consultant_associate`) —
  never reuse a slug.
- Company + role joined by a single hyphen.

## Path patterns

| File | Path |
|---|---|
| JD | `job-descriptions/{company_slug}-{role_slug}.md` |
| Company notes (one per company) | `job-descriptions/{company_slug}-company_notes.md` |
| Pass 1 analysis | `resume-outputs/p01/{company_slug}-{role_slug}_analysis.md` |
| Pass 2 resume draft | `resume-outputs/p02/{company_slug}-{role_slug}_draft.md` |
| Pass 2 cover letter draft | `resume-outputs/p02/{company_slug}-{role_slug}_cover_letter_draft.md` |
| Pass 2 changelog | `resume-outputs/p02/{company_slug}-{role_slug}_changelog.md` |
| Pass 3 final resume | `resume-outputs/p03/{company_slug}-{role_slug}_final.md` |
| Pass 3 final cover letter | `resume-outputs/p03/{company_slug}-{role_slug}_cover_letter_final.md` |
| Pass 3 tailoring notes | `resume-outputs/p03/{company_slug}-{role_slug}_tailoring_notes.md` |
| Final PDF (Josh generates in Google Docs) | `Ekong_Joshua_Resume_{CompanyShortName}_{Year}.pdf` |

`{CompanyShortName}` for the PDF is a short human-readable name or abbreviation
(e.g. `BCG`), not the underscore slug. Pass 3 suggests this filename in its Build
spec — the PDF itself is never generated by the agent.

## Schemas

**JD file:**
```
---
Company: {Full Company Name} ({Abbreviation if applicable})
Role: {Exact Role Title}
Source: {URL}
Date pulled: {YYYY-MM-DD}
---

{full JD text}
```

**Company notes file** (matches Josh's existing files exactly — do not deviate):
```
---
Company: {Full Company Name} ({Abbreviation if applicable})
Role(s) Applied:
- {Role title 1}
- {Role title 2}
Sources:
- {URL 1}
- {URL 2}
---

{free-form notes; "None" if empty}
```
